# SpartaExam Studio 사용자/관리자 가이드

이 문서는 실제 운영 기준으로 자주 쓰는 흐름만 정리했습니다.

## 공통 시작
1. 서비스에 로그인합니다.
2. 권한에 따라 화면이 달라집니다.
- 관리자: 시험 생성/채점/운영 메뉴 표시
- 수강생: 응시 가능한 시험 목록 중심

## 수강생 가이드
### 시험 응시
1. `문제 풀이`에서 시험을 선택합니다.
2. 시험 화면에 들어오면 타이머가 시작됩니다.
3. 새로고침/재접속해도 남은 시간은 이어집니다.
4. 답안 작성 후 `시험지 제출`을 누르고 확인 모달에서 확정합니다.

### 제출 후 확인
- 제출 완료 후에는 재응시가 불가능합니다.
- `내 제출`에서 제출 답안, 정답 기준, 채점 상태를 확인할 수 있습니다.

### 점수가 안 보일 때
- 보통 관리자 결과 공유 전 상태입니다.
- 채점 완료여도 공유 전에는 사용자 화면에서 숨김 처리될 수 있습니다.

## 관리자(튜터) 가이드
### 시험 생성/수정
1. `/admin/problems` 또는 `/admin/exams`로 이동
2. 시험 메타(제목, 설명, 트랙, 시간) 입력
3. 문항 추가(객관식/주관식/코딩)
4. 정답/채점 기준 입력 후 저장
5. 필요하면 리소스(데이터/실습 파일) 업로드

### 채점 운영
1. `/admin/grading`에서 자동 채점 실행
2. `/dashboard`에서 문항별 결과 확인
3. 필요 시 수동 채점/재채점으로 보정

### 결과 공개
1. `/admin/exams`에서 대상 시험 선택
2. `유저 공유`로 공개
3. 필요 시 다시 비공개 전환

### 사용자 관리
- 수강생 계정 조회/삭제 가능
- 관리자 계정은 삭제 대상 제외

## 운영 체크(간단 버전)
- 시험 시작 전: 문항/정답/시간/리소스 확인
- 시험 종료 후: 자동 채점 실행, 실패 건 수동 보정
- 결과 공지 전: 공유 상태 + 학생 계정에서 노출 확인

## 자주 묻는 질문
### Q. 브라우저를 닫으면 시간이 멈추나요?
아니요. 서버 시간 기준이라 계속 흐릅니다.

### Q. 제출 후 다시 풀 수 있나요?
아니요. 재응시는 막혀 있습니다.

### Q. 채점 결과가 안 보입니다.
관리자가 아직 결과를 공유하지 않았을 가능성이 큽니다.

## 문제 생겼을 때 순서
1. 로그인/권한 확인
2. 시험 상태(`published`) 확인
3. 결과 공유 여부 확인
4. 필요하면 관리자 감사 로그/헬스체크 확인
