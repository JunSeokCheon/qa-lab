# QA 서비스 사용자/관리자 가이드

## 1) 공통 시작
1. 서비스 접속 후 계정으로 로그인합니다.
2. 계정 권한에 따라 화면이 다릅니다.
- 관리자: 관리자 기능 메뉴 노출
- 수강생: 응시 가능한 시험 목록 중심 화면

## 2) 수강생 가이드
### 시험 응시
1. `문제 풀이` 또는 시험 목록에서 시험을 선택합니다.
2. 시험 화면 진입 순간 타이머가 시작됩니다.
3. 브라우저를 닫거나 새로고침해도 남은 시간은 계속 줄어듭니다.
4. 답안을 작성한 뒤 페이지 하단 `시험지 제출` 버튼을 누릅니다.
5. 확인 박스에서 `제출`을 확정하면 최종 제출됩니다.

### 제출 후
- 같은 시험은 다시 응시할 수 없습니다.
- `내 제출` 화면에서 다음을 확인할 수 있습니다.
  - 내가 제출한 답안
  - 문항 정답(설정값)
  - 채점 결과/피드백(공유된 경우)

### 점수 확인
- 관리자가 결과 공유를 눌러야 최종 점수가 표시됩니다.
- 공유 전에는 채점 완료 상태여도 결과가 숨겨질 수 있습니다.

## 3) 관리자(튜터) 가이드
### 시험 만들기
1. 관리자 > 시험지 관리로 이동
2. 시험 제목/설명/트랙/시험 시간(분) 입력
3. 문항 유형(객관식/주관식/코딩) 추가
4. 각 문항의 정답 입력
5. 저장 후 필요 시 리소스(데이터/실습 파일) 업로드

### 채점 운영
1. 자동 채점 관리에서 `자동 채점 시작!` 승인
2. 시험 대시보드에서 문항별 채점 결과 확인
3. 이의제기/오채점은 수동 채점으로 보정

### 결과 공개
1. 관리자 > 시험 목록 관리
2. 대상 시험 선택
3. `유저 공유` 버튼으로 결과 공개
4. 필요 시 `공유 해제` 가능

### 사용자 관리
- 사용자 관리 페이지에서 수강생을 조회/삭제할 수 있습니다.
- 관리자 계정은 보안상 삭제 대상에서 제외됩니다.

## 4) 운영 체크리스트 (비개발자용)
- 시험 시작 전
  - 시험 시간/문항 정답/리소스 업로드 확인
- 시험 종료 직후
  - 자동 채점 시작 승인
  - 실패 문항 수동 채점 여부 점검
- 결과 공지 시점
  - 유저 공유 버튼 실행
  - 표본 학생 계정으로 결과 노출 확인

## 5) 자주 묻는 질문
### Q1. 시험 중 브라우저를 닫으면 시간이 멈추나요?
아니요. 서버 기준으로 시간이 흐르므로 재접속해도 남은 시간만 표시됩니다.

### Q2. 제출 후 다시 풀 수 있나요?
아니요. 제출 완료 시 재응시가 차단됩니다.

### Q3. 채점 결과가 안 보입니다.
관리자가 아직 결과 공유를 하지 않았을 수 있습니다.

## 6) 문제 발생 시 우선 점검
1. 로그인 상태 확인
2. 시험 상태(`published`) 확인
3. 관리자 결과 공유 여부 확인
4. 그래도 안되면 관리자 감사 로그와 서버 헬스체크 확인
